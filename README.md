# Treinando_Pandas

É um pequeno projeto que fiz dentro do curso de "Análise e Visualização de Dados com Pandas & SQL" da Asimov Academy.

Caso tenha interesse em conhecer a plataforma, segue o link: [Asimov Academy](https://asimov.academy/?utm_source=hub&utm_medium=site&utm_campaign=relacionamento&utm_content=header)

Salvei uma versão do projeto em formato HTML, para visualizar o projeto sem precisar de um ambiente de desenvolvimento.
